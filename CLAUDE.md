# jobfish.ing

Job application tracking and hunting tool.

## MCP Integrations

- **GitHub** (`plugin:github`) — repo: `Luc-AI/jobfish.ing`
- **Supabase** — database & auth backend
- **Vercel** — deployment
- **PostHog** — product analytics
- **shadcn** — component library
- **Sentry** — error monitoring
- **Resend** — transactional email (needs `RESEND_API_KEY` env var)

## Dev Setup

```bash
# Install dependencies
npm install

# Run dev server
npm run dev
```
